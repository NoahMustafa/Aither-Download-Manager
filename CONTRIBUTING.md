# Contributing to Aither

Aither's source is proprietary, so code contributions (PRs changing app
behavior) aren't accepted directly — but bug reports, feature requests, and
translation feedback are very welcome and directly shape the roadmap.

## Reporting a bug

Open a [bug report](https://github.com/NoahMustafa/Aither-Download-Manager/issues/new?template=bug_report.yml).
Include:

- Aither version (Settings → About)
- Windows version
- What you expected vs. what happened
- Steps to reproduce, and logs if relevant (`%APPDATA%\Aither\logs\` — logs
  are encrypted and secrets are redacted, safe to attach)

## Requesting a feature

Open a [feature request](https://github.com/NoahMustafa/Aither-Download-Manager/issues/new?template=feature_request.yml).
Check the [Roadmap](README.md#roadmap) first — it may already be planned.

## Translations

Aither ships 16 languages. Found a wrong or awkward translation string? Open
an issue with the language, the string, and the correction.

## Questions / discussion

Use [Discussions](https://github.com/NoahMustafa/Aither-Download-Manager/discussions)
for anything that isn't a bug or a concrete feature request.
